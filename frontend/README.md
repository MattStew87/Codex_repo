# Next.js Frontend

This is a minimal Next.js application that pairs with the Python backend. It includes a single page and an example API route.

## Setup

Install dependencies from the project root:
```bash
cd frontend
npm install
```

## Running the development server

Start the Next.js dev server on port 3000:
```bash
npm run dev
```
Visit `http://localhost:3000` to view the frontend.
